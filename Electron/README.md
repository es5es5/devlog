## 3. Vue 와의 궁합.

> JavaScript is EVERYWHERE !

&nbsp;&nbsp;최근 웹개발 트렌드가 서버에서 클라이언트 쪽으로 대부분의 비즈니스 로직이 넘어오면서. 프론트엔드 프레임워크들의 위상이 매우 높아졌다. (React, Vue, ~~Angu어쩌구~~ 그리고 Svelte)
